\# 👾 Space Invader Game



A simple 2D space shooter game developed in Java using JavaFX.



\## 🕹️ Project Overview



This project is a classic arcade-style space game where the player controls a ship, moves left and right to shoot incoming enemies, collects valuable rewards, and carefully avoids penalties. 



The game loop is fully featured, including pause and restart mechanics, real-time score tracking, dynamic enemy spawning, power-ups like enhanced fire, and a complete main menu to game-over flow.



\## 🛠️ Technologies



\* Java

\* JavaFX



\## ✨ Features



\* Smooth player movement and responsive shooting system

\* Dynamic enemy spawning and score tracking

\* Randomized reward and penalty drops

\* Enhanced fire power-up system

\* Pause and restart functionality

\* Complete UI flow: Main Menu and Game Over screens



\## 🎮 Controls



\* \*\*LEFT / RIGHT Arrows\*\*: Move player

\* \*\*SPACE\*\*: Shoot

\* \*\*P\*\*: Pause the game

\* \*\*R\*\*: Restart the game

\* \*\*ESC\*\*: Return to main menu or exit



\## 🚀 How to Run



This project is a JavaFX desktop application.



To play the game, compile and run the `SpaceInvader.java` file in any standard IDE or environment configured with JavaFX support.
